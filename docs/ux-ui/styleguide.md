# Guia de Estilos e UX (Fluent 2 e A11y)

Este projeto adere rigidamente ao **Microsoft Fluent 2 Design System**.

## Cores e Tema
- Tema dinâmico (Light/Dark mode) com suporte nativo pelos Web Components.
- Acentos visuais utilizando a paleta oficial do Fluent UI.

## Acessibilidade (A11y)
- Contraste WCAG 2.1 AA obrigatório.
- Navegação por teclado suportada em 100% dos fluxos.
- Suporte a leitores de tela nativo via Aria-Labels.

## UX Principles
- **Micro-interações:** Respostas visuais claras (hover, active, focus).
- **Glassmorphism:** Uso sutil de profundidade e desfoque para separar contextos modais do fundo da aplicação.
